# Remote Laboratory by IMASS

Remote Laboratory by IMMAS is a project where it will be implemented a way to make laboratory experiments from your house through internet, futhermore using real hardware.

## Folder Structure 

1. Frontend
 -  Web
 - Workspace
2. Backend
 - Web
 - Database
 - Orchestrator 
3. Hardware
 - Boards
 - Actuators
 - Lights
 - Sensors
 - Power
 - Net
